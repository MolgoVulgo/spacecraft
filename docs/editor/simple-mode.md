# Editor simple mode

Simple mode is the baseline authoring mode for catalog variants.

Main behaviors:

- voxel-oriented preview
- face selection on 1x1x1 cells
- anchor editing from selected faces
- round/chamfer operations from selected voxel faces
- specs and recipes through modals

Simple mode does not expose point, edge, or draft-face editing.

Use simple mode when:

- the shape is still mostly box/voxel based
- anchor placement matters more than custom planar edits
- you want the shortest path to a valid catalog variant
