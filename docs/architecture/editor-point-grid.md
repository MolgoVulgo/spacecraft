# Editor Point Grid

## Phase 5 scope

- `src/3d/editor/editorPointGrid.js`
  - génération des points 0.5 dans la bounding box officielle
  - validation simple des bounds
  - résumé de grille pour l’UI avancée

## Current behavior

- les points sont affichés uniquement en mode avancé
- les points surface sont plus visibles que les points internes
- la grille dépend uniquement de la taille active et du `subgrid_unit`
- aucun point n’est écrit dans le catalogue

## Non-goal

- aucune sélection de point n’est encore active dans cette phase.
