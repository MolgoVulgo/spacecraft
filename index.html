Symétrie
Variante
Scène vide. Double-clique une taille ou glisse une taille dans la scène.
  • Ajouter une pièce : double-clic sur une taille ou glisser-déposer vers la scène
  • Appliquer une forme : cliquer une variante avec une pièce déjà sélectionnée
  • Clic gauche sur une pièce ou un groupe : sélectionner puis glisser pour déplacer
  • Clic gauche sur le fond : efface la sélection ; glisser sur le fond : rectangle de sélection
  • Shift + clic : ajoute ou retire une pièce ou un groupe de la sélection
  • Clic droit maintenu + glisser : déplacement caméra ; molette : zoom
  • W/S : profondeur, A/D : horizontal, Q/E : vertical
  • Clavier AZERTY : Z/S profondeur, Q/D horizontal, A/E vertical
  • Les touches de déplacement déplacent la sélection, ou la scène si rien n’est sélectionné
  • Ctrl/Cmd + C : copier ; Ctrl/Cmd + V : coller
  • Ctrl/Cmd + G : créer groupe ; Ctrl/Cmd + B : dissoudre groupe sélectionné
  • Ctrl/Cmd + Z : annuler déplacement ; Ctrl/Cmd + Y ou Ctrl/Cmd + Shift + Z : refaire
  • F : cadrer la sélection, sinon toute la scène ; + : cadrer la sélection
  • R : réinitialiser la vue ; Échap : vider la sélection ; Suppr / Retour arrière : supprimer la sélection
