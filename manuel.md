# Manuel

## Assembly

### Ajout et sélection

- Ajouter une pièce : double-clic sur une taille dans le panneau catalogue, ou glisser-déposer vers la scène.
- Sélectionner une pièce ou un groupe : clic gauche.
- Sélection multiple : `Shift + clic` ajoute ou retire un élément de la sélection.
- Rectangle de sélection : cliquer-glisser sur le fond de scène.
- Vider la sélection : clic gauche sur le fond sans glisser, ou `Échap`.

### Déplacement

- Glisser une pièce ou un groupe : clic gauche maintenu sur l’élément sélectionné.
- Déplacement clavier QWERTY :
  - `W` / `S` : profondeur `+Y / -Y`
  - `A` / `D` : horizontal `-X / +X`
  - `Q` / `E` : vertical `-Z / +Z`
- Déplacement clavier AZERTY :
  - `Z` / `S` : profondeur `+Y / -Y`
  - `Q` / `D` : horizontal `-X / +X`
  - `A` / `E` : vertical `-Z / +Z`
- Si une sélection existe, ces touches déplacent la sélection.
- Si rien n’est sélectionné, ces touches déplacent la scène.

### Caméra

- Déplacement caméra : clic droit maintenu + glisser.
- Zoom : molette.
- Cadrer :
  - `F` : cadre la sélection, sinon toute la scène.
  - `+` ou pavé numérique `+` : cadre la sélection.
- Réinitialiser la vue : `R`.

### Groupes

- Créer un groupe : `Ctrl/Cmd + G`.
- Dissoudre le groupe sélectionné : `Ctrl/Cmd + B`.

### Édition

- Copier la sélection : `Ctrl/Cmd + C`.
- Coller : `Ctrl/Cmd + V`.
- Supprimer la sélection : `Suppr` ou `Retour arrière`.

### Historique

- Annuler un déplacement : `Ctrl/Cmd + Z`.
- Refaire : `Ctrl/Cmd + Y`.
- Refaire aussi : `Ctrl/Cmd + Shift + Z`.

## Notes

- Les raccourcis clavier sont ignorés quand un champ texte, une liste ou une zone de saisie a le focus.
- L’historique actuel couvre les déplacements, y compris les déplacements par glisser et au clavier.
